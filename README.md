# (Pre)ScoutingDataCompiler

A project to compile data from [Statbotics](https://www.statbotics.io/api/rest), [TBA](https://www.thebluealliance.com/apidocs/v3), and the [FMS API](https://frc-api-docs.firstinspires.org/) into a single utile form. Also working on DataVis and integrating scouting data. Using Pandas and OpenPyXL.
